# DiscvrAI — CMS Infosystems Dynamic Demo Spec (v2)

Three separate full-screen demos · Dynamic execution · No client name references

## Overview

Build a professional operations intelligence demo app called "DiscvrAI – Agentic Operations Platform". This is a live demo for a senior executive audience (Puneet Bhirani, COO at CMS Infosystems). All data is pre-defined — no backend required.

## App Flow — Five Screens

- [0] Intro / Landing
- [1] Demo 1 — Financial Reconciliation (Margin Leakage Recovery + Decision Intelligence)
- [2] Demo 2 — Integration & Order Execution (Revenue Unblock)
- [3] Demo 3 — Route & Dispatch (Agentic Route & Dispatch Logic)
- [4] Summary / Next Steps

## Key Design Rules

- Generic "X Company" — no hardcoded real company names
- Dark theme: #0A0F1E background, #111827 cards, indigo accent (#6366F1)
- Agent badge with ⚡ prefix
- Framer Motion animations throughout
- All data pre-computed in /src/data/ files

## Demo Script (12-14 min total)

1. Opening (1 min) — Intelligence Gap framing
2. Demo 1 — Reconciliation (4-5 min) — Run agent, review matches, approve exception
3. Demo 2 — Integration (3-4 min) — View pipeline, run tests, view failures
4. Demo 3 — Route & Dispatch (3-4 min) — Scan fleet, approve actions
5. Summary (1 min) — Three pillars recap + pilot proposal

## Three Pillars Alignment

- Margin Leakage Recovery → Demos 1 + 2
- Decision Intelligence → Demos 1 + 2 + 3
- Agentic Route & Dispatch Logic → Demo 3

# DiscvrAI: Presentation Deck Structure
## Flexible Deck for Startup Expo & Enterprise Meetings

---

## DECK ARCHITECTURE: MODULAR APPROACH

**Core Slides (1-10):** Universal story - works for all audiences  
**Industry-Specific Slides (11-15):** AMC, Manufacturing, Healthcare  
**Closing Slides (16-20):** Universal call-to-action

**Total Slides:** 20-25 slides  
**Timing for Backdrop:** 15-20 seconds per slide (auto-advance)  
**For Client Meetings:** Manual control, 15-20 minutes presentation

**Note:** This deck works for both expo backdrop (auto-advancing) and client meetings (manual control)

---

## HOW TO USE THIS DECK

### For Expo Backdrop (Auto-Advancing Loop):
1. **Setup**: Configure auto-advance mode (15-20 seconds per slide)
2. **Rotation Strategy**: 
   - **Option A (Recommended)**: Show all industries in sequence
     - Slides 1-10 (Core) → Slides 11-15 (AMC) → Slides 11-15 (Manufacturing) → Slides 11-15 (Healthcare) → Slides 16-20 (Closing)
   - **Option B**: Rotate by industry (change every 5-10 minutes)
     - AMC rotation: Slides 1-10 + AMC 11-15 + Closing
     - Manufacturing rotation: Slides 1-10 + Manufacturing 11-15 + Closing
     - Healthcare rotation: Slides 1-10 + Healthcare 11-15 + Closing
3. **Visual Requirements**: 
   - Large text (minimum 24pt font)
   - High contrast colors
   - Clear, simple graphics
   - Readable from 10-15 feet away
4. **Team Interaction**: Team members can pause/explain specific slides when visitors show interest

### For Client Meetings (Manual Control):
1. **AMC Clients**: Use slides 1-10 + AMC-specific slides (11-15) + Closing (16-20)
2. **Manufacturing Clients**: Use slides 1-10 + Manufacturing slides (11-15) + Closing (16-20)
3. **Healthcare Clients**: Use slides 1-10 + Healthcare slides (11-15) + Closing (16-20)
4. **Other Industries**: Use slides 1-10 + Enterprise template (11-15) + Closing (16-20)
5. **Timing**: 15-20 minutes presentation + 20-30 minutes discussion
6. **Customization**: Add client-specific use cases, metrics, or examples where relevant

## PRESENTATION TIPS

### For Expo Backdrop (Auto-Advancing):
- **Timing**: 15-20 seconds per slide (total loop: 5-7 minutes)
- **Setup**: Auto-advance mode, continuous loop
- **Visuals**: Large, clear text and graphics (readable from distance)
- **Focus**: Show all industries (AMC, Manufacturing, Healthcare) in rotation
- **Team**: Team members can pause/explain specific slides when visitors show interest

### For Client Meetings (Manual Control):
- **Time**: 15-20 minutes presentation + 20-30 minutes discussion
- **Focus**: Business outcomes, integration, ROI
- **Tone**: Professional, consultative
- **Visuals**: 
  - **AMC Clients**: Use slides 1-10 + AMC-specific slides (11-15)
  - **Manufacturing**: Use slides 1-10 + Manufacturing slides (11-15)
  - **Healthcare**: Use slides 1-10 + Healthcare slides (11-15)
  - **Other Industries**: Use slides 1-10 + Enterprise slides (11-15)

### General Tips:
- **Start with Problem**: Always lead with pain points
- **Show, Don't Tell**: Use visuals, diagrams, metrics
- **Be Specific**: Avoid generic statements - use numbers, examples
